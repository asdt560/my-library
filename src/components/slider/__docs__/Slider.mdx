# Slider

Slider component with different props.

#### Example

<Canvas of={Slider.Primary} />

## Usage

```ts
import {Slider} from "sld-ui";

const Example = () => {
  return (
      <Slider
        max={100}
        min={0}
        step={0.01}
      />
  );
};

export default Example;
```

#### Arguments

- **min** - A numerical value that represents the minimal amount of the slider.
- **max** - A numerical value that represents the maximal amount of the slider.
